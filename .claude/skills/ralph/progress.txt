## Codebase Patterns
- i18n: All UI strings go in both `src/web/src/i18n/locales/en.json` and `pt.json` тАФ nested keys under feature namespaces
- Typecheck: `cd src/web && npx tsc -b` must pass before committing
- Frontend tests: `cd src/web && npm test` to run all tests
