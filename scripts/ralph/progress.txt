## Codebase Patterns
- .NET 8 Web API project lives at `src/api/GeekVault.Api/`
- DbContext is `ApplicationDbContext` in `Data/ApplicationDbContext.cs`
- Connection string key: `DefaultConnection` in appsettings.json
- SQL Server: `localhost,1433`, user `sa`, password `P@ssw0rd`, database `ApplicationDB`
- Uses minimal API style (not controllers)
- Build with `dotnet build` from the project directory
- DbContext inherits from `IdentityDbContext<ApplicationUser>` — always call `base.OnModelCreating()` in OnModelCreating
- Identity registered with `AddIdentity<ApplicationUser, IdentityRole>()` in Program.cs
- Migrations generated with `dotnet ef migrations add <Name>` from the project directory
- Use `OwnsMany` + `ToJson()` for JSON columns storing lists of owned types in EF Core 8
- Entity models live in `Models/` directory under namespace `GeekVault.Api.Models`
